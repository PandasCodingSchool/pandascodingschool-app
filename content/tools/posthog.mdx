## Overview

PostHog is an open-source product analytics platform that combines analytics, session recordings, feature flags, A/B testing, and surveys in a single product. It's become the go-to choice for developer-focused teams who want control over their data.

## Key Features

### Product Analytics
Track events, build funnels, analyze user paths, and create dashboards. PostHog's query engine is powerful and the UI is developer-friendly.

### Session Recordings
Watch real user sessions to understand behavior and debug issues. No additional tool needed.

### Feature Flags
Ship features safely with percentage rollouts, user targeting, and instant kill switches. Feature flags are tightly integrated with analytics.

### A/B Testing
Run experiments with statistical significance tracking. Results are automatically connected to your analytics data.

## Why Developers Love PostHog

PostHog was built for developers. The SDK is clean, the API is well-documented, and the self-hosting option gives you full control over your data. The generous free tier (1 million events per month) means you can get started without a credit card.

## Who Should Use PostHog

Any product team that wants a unified analytics platform without vendor lock-in. Particularly strong for startups and developer tools companies.
